## Numbers
- Immutable

```py
abs(x)
pow(x, y)
round(x)

max(x, y, ...)
min(x, y, ...)
sum(seq)
```

## Math
```py
import math
math.sqrt(x)
math.pow(x, y)
math.ceil(x)
math.floor(x)
```

## Random
```py
import random
random.random() # Random float between 0 and 1
random.randrange(start, stop, step) # Random element between start & stop
random.choice(seq) # Random item from a list, tuple or string
```
