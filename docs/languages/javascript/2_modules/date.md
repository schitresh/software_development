## Date
```js
new Date() // Current date & time
new Date(datestring)
new Date(year, month, date)
new Date(year, month, date, hour, minute, second, millisec)
new Date(epoch_secs)
```

### Instance Methods
```js
// 0 is Sunday
getDay() // getDate, getMonth, getYear, getFullYear
getHours() // getMinutes, getSeconds
setDay() // setDate, setMonth, setYear, setFullYear
setHours() // setMinutes, setSeconds

getUTCDay() // getUTCDate, getUTCMonth, getUTCYear, getUTCFullYear
setUTCDay() // setUTCDate, setUTCMonth, setUTCYear, setUTCFullYear

toString() // toISOString, toUTCString
toDateString() // Thu Apr 12 2018
toTImeString()
```
