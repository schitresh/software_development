## Numbers
```js
toFixed(x) // Shows x digits after decimal
toPrecision(x) // Shows x digits including both before and after decimal
toString()

isInteger()
Number()
parseInt()
parseFloat()
```

## Random
```js
Math.random() // Random float between 0 and 1
min + Math.random() * (max - min) // Random element between start & stop
```
